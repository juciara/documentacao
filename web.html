Atenção

  • CSS

    • O projeto utiliza o [bootstrap-fatec](https://github.com/fatechub/fatec-bootstrap), toda alteração de css deve ser feita na pasta `assets/css/estilo.css`, **nunca** alterar o arquivo bootstrap direto.

  • HTML

    • Pelo projeto ser feito utilizando **PHP**, é usado o conceito de MVC, onde a parte de visualização é separada da regra de negócio. É também interessante pelo fato de que um menu pode ser inserido nas páginas através de um unico arquivo .html, facilitando manutenção e escalabilidade. [MVC]() , [Views PHP]()
